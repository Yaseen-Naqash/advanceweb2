        4- makemigrations and migrate

    AUTHENTICATING USERS USING PREMADE FUNCTIONS
            0-  from django.contrib.auth import authenticate, login, logout
                from django.contrib.auth.decorators import login_required

            1- in your settings.py add :
                LOGIN_URL = 'my_login_url'


            2- you need login_function() , logout_function()

            3- def login_function():
                    user = authenticate(username='someusername', password='somepassword')
                        if user is not None:
                            login(request, user)
                        else:
                            print('some error messages')
                            return redirect('login_url_name')

            4- def logout_function():
                logout(request)
                return redirect('login_url_name')

            5- add these urls :
                    path('login/',login_function, name='my_login_url'),
                    path('logout/',logout_function , name='my_logout_url'),
